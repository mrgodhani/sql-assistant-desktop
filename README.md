# sql-assist-desktop

An AI-powered SQL assistant desktop application. Connect to databases, ask questions in natural language, and get SQL queries generated by AI. Execute queries, view results in tables and charts, and export to Excel or CSV.

## Features

- **Multi-database support**: PostgreSQL, MySQL, SQLite, SQL Server
- **AI chat**: Natural language to SQL with OpenAI, Anthropic, Google, OpenRouter, or local Ollama
- **Schema context**: AI receives your database schema for accurate query generation
- **Results**: Sortable, filterable tables with virtual scrolling for large result sets
- **Charts**: Bar, line, pie, scatter, and area charts from query results
- **Export**: Excel (with embedded charts) and CSV
- **Persistent conversations**: Chat history saved locally

## Prerequisites

- Node.js 18+
- npm or pnpm

```bash
npm run build:linux
```

### Building for macOS (universal + notarized)

The Mac build produces a universal DMG (Intel + Apple Silicon) and notarizes it.

**Prerequisites:**
- Xcode Command Line Tools: `xcode-select --install`
- Apple Developer account
- App-specific password: [appleid.apple.com](https://appleid.apple.com) → Sign-In and Security → App-Specific Passwords

**Setup:**
1. Copy `.env.example` to `.env`
2. Set `APPLE_ID`, `APPLE_APP_SPECIFIC_PASSWORD`, `APPLE_TEAM_ID` (Team ID from [developer.apple.com](https://developer.apple.com/account))
3. Run `npm run build:mac`

Output: `dist/sql-assist-desktop-1.0.0.dmg`

On first launch from outside the Applications folder (e.g., Downloads or DMG), the app will prompt to move to Applications.

### Test

## Project Structure

- `src/main/` — Electron main process (services, IPC handlers, database)
- `src/preload/` — Preload scripts exposing secure APIs via contextBridge
- `src/renderer/` — Vue 3 frontend (components, stores, views)
- `src/shared/` — Shared types and constants
- `aidlc-docs/` — Design and construction documentation
